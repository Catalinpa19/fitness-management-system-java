# Fitness Management System (Java)

A console-based application for managing a fitness center, built to practice object-oriented programming concepts.

## ✨ Features
- Manage trainings, trainers, and subscriptions
- Different training types (Yoga, Pilates, CrossFit)
- Standard and premium subscriptions
- Trainer types (permanent and external)
- Financial report (revenue, costs, profit)

## 🛠 Technologies
- Java

## 🧠 Concepts Used
- Inheritance and polymorphism
- Abstraction
- Enums and factory-like patterns
- Basic data management

## 📌 Notes
This project was built for learning purposes to understand OOP design in Java.
